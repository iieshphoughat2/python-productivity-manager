# Python Productivity Manager

A productivity tool built using Python to manage daily tasks and expenses.

## Features
- Add and manage tasks by date
- Mark tasks as completed
- Delete tasks
- Add and track daily expenses
- View expenses by date
- Calculate total monthly expenses
- Data stored using JSON (persistent storage)

## Tech Used
- Python (functions, dictionaries, error handling)
- JSON (data storage)

## How to Run
Run the Python file:
python productivity_manager.py
